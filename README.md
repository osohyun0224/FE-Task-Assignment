## 🧑🏻‍💻 How to Setup

1. 리포지토리를 클론하고 클론한 디렉토리에 들어갑니다.

```bash
git clone https://github.com/osohyun0224/FE-Task-Assaginment
cd FE-Task-Assaginment
```

2. 다음의 명렁어를 실행하여 프로젝트를 실행합니다.

```cmd
npm install
npm run dev
```

3. 해당 프로젝트에서 사용한 패키지 버전은 다음과 같습니다. 

- Next.js ```12.1.0```
- React.js ```17.0.2```
